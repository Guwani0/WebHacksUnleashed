Welcome to the Vulnerable Web App

Explore various web vulnerabilities in a safe environment.

This intentionally vulnerable web application is designed for educational purposes. It allows users to explore and exploit common web vulnerabilities in a controlled, safe, and legal environment. Each section focuses on a specific security flaw—like CSRF, XSS, IDOR, and Broken Access Control—providing hands-on experience for aspiring ethical hackers, developers, and cybersecurity learners. Sharpen your skills, learn to think like an attacker, and understand how to defend real-world applications.
